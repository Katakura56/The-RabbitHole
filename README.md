## サービス概要
　本サービスは「行き先の詳細がわからない外部リンク」を利用者に提供するものです。あえて行き先の詳細を明かさないことで利用者の好奇心を刺激し、リンクをクリックした方にはgoogleの生成AI Gemini が選抜した良質なサイトへ案内することで、利用者に新鮮で、面白い話題をお届けします。これにより利用者が、気軽に楽しく、多様な話題を収集できるようになることを目指します。

## このサービスへの思い・作りたい理由
　現状、インターネット利用者の多くが、「新しくて、面白い話題を探すのが難しい」という悩みを抱えていると思います。
　私は普段、新しい話題探しのためにYouTubeやXといったサービスを利用しています。しかし、そういったサービス内でコンテンツを探すために利用される「おすすめ機能」や「キーワード検索機能」は、利用者の経験や知識に関連したコンテンツを探し当てるものであり、そこから「全く新しいコンテンツ」に繋がることは難しいと思います。時間をかければ見つけられるかもしれませんが、そこまで労力をかける気にはなかなかなりません。
　そういった問題の解決のために、私はこのサービスを作りたいと思いました。

## ユーザー層について
1. アルゴリズムの「フィルターバブル」に飽きた層
現代のSNSや検索エンジンは、ユーザーの好みに合わせすぎるあまり、似たような情報ばかりが表示される「フィルターバブル」という現象を引き起こしています。
* ターゲット: 毎日同じようなニュースや投稿ばかりが流れてくることに、退屈や閉塞感を感じているデジタルネイティブ世代。
* 選んだ理由: 自分の嗜好の外側にある「質の高い未知の情報」に触れることで、凝り固まった視点をリセットしたいという欲求を持っているためです。

2. インスピレーションを求めるクリエイター・企画職
デザイナー、ライター、プランナーなど、常に新しいアイデアを必要とする職種の人々です。
* ターゲット: 仕事の合間に気分転換を兼ねて、全く異なる分野の知識や視点を取り入れたいプロフェッショナル。
* 選んだ理由: 創造性は、一見無関係な情報の組み合わせから生まれることが多いためです。AIが選んだ良質なサイトであれば、単なる時間の浪費ではなく、有益な刺激（インプット）として受け入れられます。

3. 知的好奇心が旺盛な「教養・雑学好き」
特定の目的がなくても、新しいことを知ること自体を楽しみとする層です。
* ターゲット: 読書家やニュースサイトの隅々までチェックするような、知的好奇心の強い中高年〜若年層。
* 選んだ理由: 「行き先がわからない」というワクワク感そのものが、知的なゲームやクイズのような楽しみを提供します。Geminiによる「良質なサイトの選抜」が、情報の信頼性を求めるこの層のニーズと合致するためです。

4. 「スキマ時間」を豊かにしたい忙しい現代人
移動中や休憩中など、数分間の暇つぶしを求めている人々です。
* ターゲット: 通勤電車やランチタイムに、スマホで手軽に「何か面白いもの」を見つけたい会社員や学生。
* 選んだ理由: 自分で検索して探す手間を省き、ワンクリックで「当たり（良質なコンテンツ）」にたどり着けるタイパ（タイムパフォーマンス）の良さが、忙しい現代人のニーズにフィットします。


## サービスの利用イメージ
　1. 外部のwebサイトに繋がる「扉（リンク）」の選択
　・ユーザーに見えるのはURLのみです。
　・これにより、先のわからない「ワクワク感」を提供します。

　2. 外部のwebサイトへ訪問
　・Geminiによって選ばれた良質なWebサイトへご案内します。

　3. 本サービスのサイトへ帰還、URLへコメント投稿
　・リンクをクリックしたユーザーはリンクに対して簡単なコメント(絵文字など)を残せるようにします。
　・これにより、他のユーザーの好奇心をさらに刺激します。また、コメントを投稿したユーザーもある種の「達成感」を
　　味わえるようにします。

## 使用する技術スタック
非ログイン状態での「二重投稿」防止と制限
　・ブラウザ識別: localStorage にブラウザ固有のUUIDを保存し、投稿時にRails側へ送る。
　・IP制限 + 指紋: ブラウザの指紋（Fingerprint）やIPアドレスをRailsのキャッシュ（Redisなど）に一定期間保持し、同一人物による連投をサーバーサイドで弾く。
　・絵文字制限: Railsのバリデーション（Regex）で、投稿内容が絵文字のみであることを保証する。

- 使用するフレームワーク 　Ruby on Rails 8.0.x　Ruby 3.4.x　React 19.x　Node.js 22.x (LTS)
- データベース　PostgreSQL
